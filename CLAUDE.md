# Правила проекта

## Общие
- Писать чистый, читаемый код без лишних комментариев
- Не добавлять лишних абстракций — только то, что нужно по ТЗ
- Каждое изменение кода — отмечать в STATE.md и PLAN.md
- Использовать сабагентов для работы: simple — для boilerplate, coder — для логики, architect — для архитектуры

## Стек
- Frontend: React + Vite (или Next.js — обсудить)
- Backend: Node.js (Express/Fastify) или Python (FastAPI)
- LLM: DeepSeek API
- Формат файлов: DOCX (Word)
- Deploy: Docker + git pull
- Сервер: Ubuntu, Caddy, домен lek.mak-o.ru, порт 12150

## Безопасность
- Один пароль для доступа (закрытая админка)
- Пароли и секреты — только в .env, в git не коммитить
- HTTPS обязателен (Caddy)

## Формат работы
- Сначала — проектирование (архитектура, стек, декомпозиция)
- Потом — реализация по плану
- После каждого цикла — обновлять STATE.md

## Язык
- Код — на английском
- Коммуникация — на русском (как пользователь)
